# Meeting Room Booking System

This project is a web-based application developed as a Computer Science NEA for the Civil Aviation Authority (CAA) in Crawley. It replaces the manual paper-based booking process with an automated system allowing staff to view, book, and manage meeting rooms in real-time.

## Requirements

- Python 3.13 or newer
- pip

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```
